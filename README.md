# LibTrade

A textbook exchange service for university students.

## Prerequisites

* Ubuntu Server 16.04+ LTS
* MySQL Server 5.7+
* Apache 2.4+

### Built with

* Bootstrap
* Google Books
* NodeJS 12.19
* NPM 6.14
  * Axios
  * Express
  * Jest
  * MySQL
  * Passport
  * TypeORM
  * Vue
